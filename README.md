# Template-matching
### Template matching using Normalized Cross Correlation    
Merits:
1. Easy to implement.
2. Fast working if the template/databse is small.

Demerits:  
1. Slower if the template/database is large.
2. Very sensitive to any kind of rotations or scale change.
